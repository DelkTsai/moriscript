MoriScript

What is MoriScript?

MoriScript is an experiment that gives JavaScript first class support for immutable data structures. As you might have guessed from the name, it does this by including Mori as part of the runtime.

How Does it Work?

It also uses Babel to transform regular JavaScript code into calls to Mori functions, which return compiled versions of Clojure's persistent data structures.

Should I Build My Startup On It?

Nope! It's just an experimental toy, designed to help others learn to write Babel plugins. It has many known edge cases and you're better off using Mori or ClojureScript in the first place.
